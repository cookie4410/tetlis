操作方法

メニュー操作

↑、↓、→、← : カーソル移動
Enter : 決定
Esc : キャンセル、戻る

ゲーム操作

（デフォルト）

← : 左移動
→ : 右移動
↓ : ソフトドロップ
↑ : ハードドロップ
A : 左回転
D : 右回転
shift : ホールド
Esc : ポーズ
