TETRiSしか勝たん！！

ゲーム説明

「TETRiSしか勝たん」モード

「テトリス(4列消し)」でしかラインが消えないモードです。
1～3列消しの場合はラインが消える代わりに所謂「お邪魔ブロック」が下からせり上がります。
その他は通常のテトリスと変わらないルールです。

ノーマルゲーム

至って普通のテトリスです。
Tスピン、横タメ、設置猶予など極力現代的なテトリスに近づけています。
10ライン消すごとにレベルが上がり、得点倍率と落下速度が上昇します。
150ライン達成でクリアです。

カスタムゲーム

ハイスコアは記録されませんが、各種項目を設定して自由に遊べます。
@TETRiSをオンにすれば「TETRiSしか勝たん」モードになります。
テトロミノ内の項目を変更することで登場するテトロミノを制限できます。
なお「TETRiSしか勝たん」ではI型テトロミノが必須です。

操作方法

メニュー操作

↑、↓、→、← : カーソル移動
Enter : 決定
Esc : キャンセル、戻る

ゲーム操作

（デフォルト）

← : 左移動
→ : 右移動
↓ : ソフトドロップ
↑ : ハードドロップ
A : 左回転
D : 右回転
shift : ホールド
Esc : ポーズ
